## Standards

- All services emit structured JSON logs
- All services use OpenTelemetry semantic conventions
- Trace/span names follow `media.*` namespace
- Metrics avoid high-cardinality labels
- No secrets in logs/metrics/traces

## Agent Rules

- Agents must not log secret values
- Agents must not expose local paths in public reports
- Agents must use structured log events
- Agents must include correlation_id in log events
