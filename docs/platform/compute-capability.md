## VM vs LXC Rules

### Use VM When

- Docker host needed
- Dokploy deployment
- Database services
- CI runners
- OpenCue/render workers
- Kernel/module isolation needed
- GPU passthrough (future)
- Stronger isolation required

### Use LXC When

- Lightweight ingress
- Simple reverse proxy
- Admin utilities
- Low-risk stateless services
- Minimal resource overhead
