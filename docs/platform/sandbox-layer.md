# Sandbox Layer Design

**Date:** 2026-07-02
**Status:** PLANNED
**Authority:** PLATFORM-SANDBOX.0

---

## Purpose

Disposable environment for experiments, comparisons, and smoke tests.

## Layer Distinction

| Layer | Purpose | Lifecycle |
|-------|---------|-----------|
| Foundation | Core infra | Long-lived |
| Shared Services | DB, storage | Long-lived |
| Workloads | Apps | Managed |
| **Sandbox** | Experiments | Disposable |
| Scratch | Quick tests | Very short |
