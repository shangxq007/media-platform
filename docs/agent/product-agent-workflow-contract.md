# Product Agent Workflow Contract

**Date:** 2026-07-07
**Status:** STABLE
**Authority:** PRODUCT-AGENT-WORKFLOW.0

## Workflow Modes

### 1. read-only (default)

- Health check
- List workspaces
- List products
- List render jobs
- Fetch product/job/artifact by ID

### 2. inspect-product

- Requires `AGENT_PRODUCT_ID`
- Fetch product detail
- Output product context summary

### 3. create-render-job (opt-in)

- Requires `AGENT_PRODUCT_WRITE=1`
- Creates render job
- Polls status
- Fetches artifact metadata
