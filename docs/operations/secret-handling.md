## Agent Rules

1. **Never print secret values**
2. **Redact tokens in reports** — use `[REDACTED]`
3. **Do not read secret files** unless explicitly assigned
4. **Do not create/rotate production credentials**
5. **Stop if task requires real secret values**
6. **Report secret exposure risk** without copying the secret
7. **Do not bypass HOOKS.0 secret checks**

## Report Redaction Rules

| Format | Use |
|--------|-----|
| `[REDACTED]` | Default — full redaction |
| `[REDACTED: token present]` | When confirming token exists |
| `[REDACTED: value omitted]` | When value is intentionally skipped |
| `***xxxx` | Last 4 chars only for non-prod tokens (rare) |

**Default is full redaction.** Last-4-chars only when explicitly needed for debugging non-production tokens.
