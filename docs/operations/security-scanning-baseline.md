## Agent Rules

1. Agents may run scans on assigned branch/worktree
2. Agents must not print secrets from scan output
3. Agents must summarize findings safely
4. Agents must not auto-fix vulnerabilities unless assigned DEPS task
5. Agents must not suppress findings without policy note
6. Scanner failures must appear in Review Packet
