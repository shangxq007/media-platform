## Validation Requirements

| Check | Description |
|-------|-------------|
| createPreviewRenderJob | API accepts valid request |
| getJobStatus | API returns job status |
| completedResponse | Response has expected shape |
| productId present | Product ID in successful response |
| previewArtifact.artifactId | Artifact ID present |
| previewArtifact.url | Safe URL, not local path |
| previewArtifact.mimeType | MIME type present |
| structuredError | Error has code + message |
| boundedStatus | Status enum values bounded |
| noProviderInternals | No provider details exposed |
| noStorageInternals | No storage details exposed |
| noSecretsInLogs | No secrets in output |
