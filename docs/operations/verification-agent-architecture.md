## Overview

Two verification agent roles provide automated runtime and cloud-side verification without modifying production services.

## 1. PVE Deployment Verification Agent

| Attribute | Value |
|-----------|-------|
| **Purpose** | Verify preview deployments on PVE infrastructure |
| **Permission Level** | Level 1 (read/verify only) |
| **Status** | PLANNED |

### Allowed Operations

- Check PVE container/VM health
- Run smoke tests against preview endpoints
- Verify application startup logs
- Check database connectivity
- Verify Flyway migration status (read-only)
- Check FFmpeg/libass availability
- Run domain test subset
- Generate verification report

### Forbidden Operations

- Modify PVE configuration
- Start/stop containers without authorization
- Access production PVE nodes
- Modify Flyway migrations
- Handle production secrets
- Run load tests
- Modify network configuration

### Stop Conditions

1. PVE node unreachable
2. Container fails to start
3. Smoke test fails twice
4. Database connection refused
5. Unexpected resource consumption

### Report Format

```
verification_id: PVE-VERIFY-{timestamp}
environment: pve-preview
target: {container/VM name}
branch: {branch}
commit: {hash}
trigger: {manual|scheduled|post-deploy}
checks: [{name, result, duration}]
result: PASS|FAIL|BLOCKED
duration: {ms}
logs_summary: {key findings}
artifacts: {report paths}
blockers: [{description}]
next_action: {recommendation}
```

## 2. Cloud Verification Agent

| Attribute | Value |
|-----------|-------|
| **Purpose** | Verify external endpoints, services, and cloud resources |
| **Permission Level** | Level 1 (read/verify only) |
| **Status** | PLANNED |
| **Execution Environment** | AWS Lambda (planned) |

### Allowed Operations

- Check endpoint health (HTTP status, response time)
- Verify dashboard/report links resolve
- Validate Review Packet schema
- Validate LikeC4 JSON export
- Check DNS/TLS/network reachability
- Verify Cloudflare R2/Pages availability
- Check GitHub PR/report links
- Run lightweight API contract checks
- Webhook validation

### Forbidden Operations

- FFmpeg rendering
- OpenCV batch analysis
- GPU inference
- Long-running integration tests
- PVE internal checks (unless secure networking designed)
- Production secret handling
- Destructive operations
- Load testing
- Automatic scaling stress tests

### Stop Conditions

1. Endpoint unreachable
2. TLS certificate invalid
3. DNS resolution fails
4. API contract violation
5. Budget alarm triggered

### Report Format

Same as PVE Verification Agent report format.
