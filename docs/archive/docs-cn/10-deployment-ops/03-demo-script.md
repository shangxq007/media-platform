# 演示脚本

> **最后更新：** 2026-05-18

## 前置条件

- 后端运行在 `http://localhost:8080`
- 前端运行在 `http://localhost:3000`
- API 密钥已配置

## 分步演示

### 第 1 步：访问应用
1. 打开 `http://localhost:3000`
2. 验证用户仪表盘加载
3. 检查导航菜单显示可用功能

### 第 2 步：创建新项目
1. 点击"新建项目"
2. 输入项目名称："演示项目"
3. 验证项目出现在列表中

### 第 3 步：打开视频编辑器
1. 点击项目打开编辑器
2. 验证编辑器布局加载（素材库、预览、时间轴、属性面板）

### 第 4 步：上传媒体
1. 点击素材库中的"上传"
2. 拖拽视频文件
3. 验证上传进度和素材出现在库中

### 第 5 步：添加片段到时间轴
1. 从素材库拖动片段到时间轴
2. 验证片段出现在时间轴轨道上
3. 点击片段进行选择

### 第 6 步：编辑片段属性
1. 选中片段后，查看属性面板
2. 调整开始时间、时长
3. 验证预览更新

### 第 7 步：添加字幕
1. 打开字幕面板
2. 点击"添加字幕"
3. 输入文本："Hello World"
4. 设置时间：00:00 - 00:05
5. 验证字幕出现在时间轴上

### 第 8 步：应用特效
1. 打开特效面板
2. 选择一个特效（如"淡入"）
3. 验证特效已应用到选中的片段

### 第 9 步：尝试演示项目
1. 打开新的编辑器标签页
2. 点击"尝试演示项目"
3. 验证演示素材填充到库中
4. 验证创建了演示时间轴

### 第 10 步：导出视频
1. 点击"导出"按钮
2. 选择一个预设（如"default_1080p"）
3. 验证显示预算估算
4. 点击"提交渲染任务"
5. 验证返回任务 ID

### 第 11 步：监控渲染任务
1. 验证任务状态显示"排队中"
2. 等待状态变为"渲染中"
3. 等待状态变为"已完成"
4. 验证显示成品

### 第 12 步：预览成品
1. 点击已完成的成品
2. 验证预览弹窗打开
3. 验证视频播放

### 第 13 步：查看能力
1. 导航到 `/me/capabilities`
2. 验证显示当前套餐
3. 验证列出可用功能

### 第 14 步：查看用量
1. 导航到 `/me/usage`
2. 验证显示已用渲染分钟数
3. 验证显示剩余配额

### 第 15 步：提交反馈
1. 导航到 `/me/feedback`
2. 选择类型："Bug"
3. 输入标题和描述
4. 点击"提交"
5. 验证成功消息

### 第 16 步：管理控制台（仅管理员）
1. 导航到 `/admin`
2. 验证管理仪表盘加载
3. 导航到 Feature Flags
4. 验证 feature flag 列表加载

### 第 17 步：Feature Flag 管理
1. 点击一个 feature flag
2. 验证 flag 详情加载
3. 切换 flag
4. 验证变更已保存

### 第 18 步：策略模拟
1. 导航到 `/admin/policies/simulate`
2. 输入租户 ID 和功能
3. 点击"模拟"
4. 验证显示决策链

### 第 19 步：分析助手
1. 导航到 `/me/analytics`
2. 输入自然语言查询
3. 验证生成了 SQL
4. 执行查询
5. 验证显示结果

### 第 20 步：错误状态
1. 触发错误（如提交无效数据）
2. 验证错误状态显示错误代码
3. 验证显示 i18n 消息
