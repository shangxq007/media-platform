# 编辑器工作台 UI

> **模块：** `frontend/`
> **最后更新：** 2026-05-18

## 概述

视频编辑器是一个 Vue 3 单页应用，提供基于时间线的视频编辑功能，包括片段管理、特效、字幕和导出。

## 状态管理

Pinia 存储管理：
- **项目存储** — 当前项目、时间线、片段、选中项
- **用户存储** — 认证、偏好设置、能力
- **UI 存储** — 面板可见性、模态框状态
- **渲染存储** — 作业状态、制品、进度
