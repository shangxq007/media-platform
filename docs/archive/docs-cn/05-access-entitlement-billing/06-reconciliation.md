# 对账

> **模块：** `billing-module`
> **最后更新：** 2026-05-18

## 概述

对账系统将内部账单记录与外部支付提供商的记录进行匹配，以确保一致性。

## 对账流程

```mermaid
graph TB
    IMPORT["导入外部发票"] --> MATCH["与内部记录匹配"]
    MATCH -->|"已匹配"| VERIFY["验证金额"]
    MATCH -->|"未匹配"| DIFF["标记差异"]
    VERIFY -->|"一致"| RECONCILED["已对账"]
    VERIFY -->|"不一致"| DIFF
    DIFF -->|"审核"| MANUAL["人工审核"]
    MANUAL --> "|已解决"| RECONCILED
    MANUAL --> "|争议"| DISPUTE["争议处理"]
```

## 对账检查

| 检查 | 描述 |
|------|------|
| 金额匹配 | 内部金额 = 外部金额 |
| 状态匹配 | 内部状态 = 外部状态 |
| 时间匹配 | 交易日期对齐 |
| 货币匹配 | 使用相同货币 |

## 差异类型

| 类型 | 描述 | 操作 |
|------|------|------|
| 金额不匹配 | 内部 ≠ 外部金额 | 标记审核 |
| 内部缺失 | 未找到外部记录 | 创建内部记录 |
| 外部缺失 | 未找到内部记录 | 调查 |
| 状态不匹配 | 状态不同 | 更新内部状态 |
